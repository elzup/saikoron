# Saikoron - カスタムルーレット

自作のルーレットを作成・管理できるPWAアプリ

## 機能

- ルーレットの作成・編集・削除・複製
- 重み付き抽選
- アニメーション付き回転
- LocalStorageによるデータ保存
- オフライン対応（PWA）

## 開発

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev

# テスト実行
npm run test

# ビルド
npm run build

# デプロイ
npm run deploy
```

## 技術スタック

- React + TypeScript
- Vite
- React Router
- vite-plugin-pwa
- Vitest
- Firebase Hosting
